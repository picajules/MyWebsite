HTML Cheat Sheet

This page details some key tags in HTML. It is important to remember that almost all tags in HTML must be explicitly opened AND closed again!

Heading Tags

These range from h1 to h6 and are used for headings and sub-headings

Paragraphs

The paragraph tag <p> is used to denote a paragraph.

Lists

Lists can be ordered (denoted by the tag <ol>):

  1. a list item (denoted by the tag <il>)
  2. another list item
  3. and another list item
Or unordered, i.e. bullet point (denoted by the tag <ul>):
  • an item in a second list (also denoted by the tag <il>)
  • and a second item in this list
  • and a third item in this list

Images

HTML can also be used to include images using the following tag:

<img src="imagename.jpg">

and the size can also be adjusted as a style attribute:

<img src="imagename.jpg" style="width:200px;height:200px;"
